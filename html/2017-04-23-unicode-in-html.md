# Unicode in HTML

HTML에서 유니코드를 사용하려면 다음과 같은 포맷을 써야 한다.

`&#nnnn;`

or

`&#xhhhh;`

`nnnn`은 10진법 유니코드 codepoint이고 `hhhh`는 16진법 유니코드 codepoint이다.

이와 같은 방식을 [Numeric charater reference](https://en.wikipedia.org/wiki/Numeric_character_reference)라고 한다.
